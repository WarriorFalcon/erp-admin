# 前后端 API 对接表（最新版 2026-04-29）

> 数据来源：`backend-new` 最新源码，含 Demo 模式和正式模式两套路由。  
> 后端统一前缀：`http://<host>/api/` 或 `http://<host>/v1/`（部分 webhook 回调用 v1）

---

## 全局说明

| 项目 | 说明 |
|------|------|
| **DEMO_MODE** | `.env` 中 `DEBUG=True` 时自动开启，Demo 路由覆盖正式路由 |
| **认证方式** | JWT Bearer Token（`Authorization: Bearer <token>`） |
| **Swagger** | `http://localhost:8000/swagger/` |
| **Redoc** | `http://localhost:8000/redoc/` |
| **API Schema** | `http://localhost:8000/api/schema/` |

## 快速联调检查单

### 接口地址配置
```
前端 .env：VITE_API_BASE_URL=http://127.0.0.1:8000
后端运行端口：8000（默认）
DEMO_MODE：DEBUG=True 时自动开 Demo 路由
```

| P0 | `/api/auth/me` | `src/api/auth.js` | 获取用户信息 |
| P1 | `/api/goods/` | `src/api/goods.js` | 商品列表 |

| P1 | `/api/orders/` | `src/api/orders.js` | 订单列表 |
| P2 | `/api/selection/calculate/` | `src/api/selection.js` | 选品引擎 |
| P2 | `/api/ai/generate-title/` | `src/api/ai.js` | AI 生成 |
