## 1. 系统概述

**辽宁跨境宝盒**是一站式跨境电商 ERP 管理系统，覆盖从货源采集到多平台铺货的完整工作流。

### 核心能力

| 模块 | 功能 |
|------|------|
| 商品采集 | 从 1688/淘宝/拼多多 抓取商品信息 |
| AI 优化 | 自动生成英文标题、卖点、描述 |
| 多平台上架 | 一键铺货至 TikTok Shop / Shopee / Amazon / Temu / SHEIN 等 |
| 订单管理 | 多平台订单聚合、状态流转、发货 |
| 库存管理 | 多仓库存同步、预警、调拨 |
| 物流追踪 | 物流单管理、轨迹查询、运费预估 |
| 利润测算 | 成本 × 定价倍率 = 自动计算利润与风险 |
| AI 助手 | 拓悦AI驱动的智能客服「小辽」 |

### 支持平台

- **货源平台**: 1688、淘宝、天猫、拼多多
- **销售平台**: TikTok Shop、Shopee、Amazon、Temu、SHEIN、Lazada、AliExpress、eBay、Wish、Mercado、Ozon、Allegro

## 2. 注册与登录

### 2.1 登录步骤

1. 打开系统地址，自动进入登录页
2. **选择模式**: 登录页可选择「🌟 小白模式」或「⚡ 资深模式」
3. **输入手机号**: 输入 11 位大陆手机号
4. **获取验证码**: 点击「获取验证码」按钮
   - 开发模式下验证码自动填充
   - 生产模式下发送真实短信
5. **输入验证码**: 填写收到的 4-6 位数字验证码
6. **点击「登录」**: 登录成功后自动跳转控制台

> **注意**: 验证码有效期为 5 分钟，60 秒内不可重复发送。勾选「记住手机号」可下次自动填充。

### 2.2 注册说明

系统不提供自主注册入口。新用户请联系管理员在后台创建账号。

### 2.3 退出登录

清除浏览器 localStorage 中的 `access_token` 或访问 `/login` 页面即可重新登录。

## 3. 双模式系统（小白模式 & 资深模式）

系统提供两种操作模式，可在登录时选择，也可登录后随时切换。

| 特性 | 🌟 小白模式 | ⚡ 资深模式 |
|------|-----------|-----------|
| 适用人群 | 新手卖家 | 多平台运营 |
| 控制台 | 简洁首页，3 个核心指标 | 完整 Dashboard，多图表 |
| 采集上货 | 3 步引导式工作流 | 全量功能面板 |
| 批量模式 | 不显示 | 支持批量采集铺货 |
| AI 提示 | 全程带路式提示 | 按需使用 |

### 切换方式

1. 登录页直接选择模式卡片
2. 小白模式控制台顶部点击「切换到资深模式」

> **注意**: 模式选择保存到浏览器，刷新不丢失。

## 4. 控制台（Dashboard）

### 4.1 小白模式控制台

简洁聚焦，三大核心区域：

- **核心指标卡片**: 今日订单、今日销售额、待发货订单
- **两大核心入口**: 「我要上货」「我要发货」
- **待办提醒**: 待发货订单数量、建议操作

### 4.2 资深模式控制台

完整数据分析面板：

- **顶部统计栏**（4 项）: 今日订单量、销售额、平均客单价、待发货订单
- **左侧图表**: 销售额趋势折线图、平台分布环形饼图
- **中间地图**: 全球订单实时分布（ECharts 世界地图，支持缩放、点击查看地区详情）
- **右侧面板**: 实时订单列表（自动刷新）、交付时效卡片
- **交付时效**: 平均交付天数、准时率、在途订单数（真实数据库统计）

> **数据来源**: 所有数据来自 `GET /api/dashboard/stats` 和数据库实时查询，非假数据。

## 5. 一站式采集上货（核心功能）

这是系统的核心工作流，导航菜单路径: **商品 → 一站式采集上货**。

### 5.1 小白模式工作流（3 步上货）

```
第1步: 粘贴货源链接 → 第2步: AI自动处理 → 第3步: 确认上架
```

**操作步骤**:

1. 复制 1688/淘宝/拼多多 商品链接
2. 粘贴到输入框（按 `Ctrl+Enter` 或点击「🚀 开始采集并上架」）
3. 系统自动完成:
   - 采集商品信息（标题、图片、价格）
   - AI 合规预检（禁售词/侵权风险检测）
   - AI 智能优化（英文标题、卖点、描述）
   - 利润测算（成本 × 定价倍率 = 建议售价）
4. 在弹出的选品决策弹窗中:
   - 查看利润数据（成本价 → 建议售价 → 预估利润）
   - 可切换定价模板（普通/活动/高端/成本加成）
   - 查看 AI 评估理由
5. 点击「确认上架」→ 商品铺货至目标平台

### 5.2 资深模式工作流

资深模式保留完整控制面板:

- **左侧**: 链接输入区 + 目标平台选择 + 一键采集按钮
- **右侧**: 商品信息预览 + AI 优化结果
- **爆品灵感推荐**: 可展开的热卖商品推荐面板

### 5.3 批量模式（资深专属）

1. 点击页面右上角「批量模式」展开
2. 选择导入方式:
   - **链接批量导入**: 每行一个商品链接
   - **Excel 文件导入**: 上传 .xlsx/.csv 文件
3. 配置铺货规则:
   - 目标平台（多选）
   - 加价方式（固定加价 / 倍率加价）
   - 店铺选择
4. 点击「开始批量采集并铺货」
5. 查看执行进度条和成功/失败统计

### 5.4 集采模式（资深专属，步骤3-5）

当使用一站式流程完成采集后，系统自动进入集采面板:

```
步骤3: 选品评估 → 步骤4: 筛选确认 → 步骤5: 一键上架
```

- **选品评估**: AI 自动对每件商品评级（推荐 / 可上架 / 不建议）
- **筛选确认**: 按利润额或利润率一键筛选，表格展示成本/售价/利润
- **一键上架**: 确认后批量提交上架，实时进度条展示成功/失败

## 6. 商品上货（ListingView）

导航菜单: **商品 → 商品上货**

### 6.1 左右双栏布局

- **左侧**: 待上货商品列表（从数据库加载）
  - 点击商品卡片即可选中
  - 支持搜索过滤
- **右侧**: AI 操作面板
  - 生成标题 / 生成卖点 / 生成描述
  - AI 文生图

### 6.2 操作步骤

1. 页面加载后自动选中第一个商品（AI 按钮解锁）
2. 点击「生成标题」→ AI 生成跨境电商英文标题
3. 点击「生成卖点」→ AI 生成 3-5 条核心卖点
4. 点击「生成描述」→ AI 生成中英文商品描述
5. 在「AI文生图」输入框填写图片描述 → 点击按钮 → AI 返回专业拍摄提示词

### 6.3 拖拽上货

选择商品卡片后可拖拽至目标平台上架区域（集采工作流）。

## 7. 商品采集（CollectView）

导航菜单: **商品 → 商品采集**

用于从 1688 批量采集商品。需要登录 1688 账号以获得最佳采集效果。

### 7.1 采集方式

- **关键词搜索**: 输入搜索词，自动从 1688 搜索
- **链接采集**: 粘贴单个商品链接
- 支持扫码枪快速采集

### 7.2 采集引擎

后端采集引擎支持:
- 1688 __PRELOADED_STATE__ JSON 解析（精准）
- CSS 选择器 + 正则匹配（通用）
- 降级策略（title/img 标签提取）
- 反屏蔽: UA 轮换（5 个）+ 同域名请求间隔 ≥ 1.5 秒

内嵌浏览器，可在 ERP 内直接浏览 1688/淘宝/拼多多。

### 9.1 功能

## 18. 常见问题（FAQ）

### Q1: 登录时收不到验证码？

- 开发模式下验证码自动填充，无需等待短信
- 生产模式请检查短信服务配置（`TUOYUE_SMS_*` 环境变量）

### Q2: AI 生成标题/描述失败？

- 检查后端是否正常运行: `http://127.0.0.1:8000/api/health/`
- 检查拓悦 AI API Key 是否配置: 环境变量 `TUOYUE_NEW_API_AUTHORIZATION`
- AI 接口超时设置为 90 秒，请耐心等待

### Q3: 采集 1688 商品失败？

- 1688 有反爬机制，部分页面需登录后才能访问
- 尝试在平台浏览器中打开商品页面后拖拽采集
- 批量采集有反屏蔽机制（UA 轮换 + 限流），但仍有被拦截风险

### Q4: 前台页面空白/按钮灰色？

- 确认后端已启动（`python manage.py runserver 8000`）
- 确认前端已启动（`cd erp-admin && npm run dev`）
- 检查浏览器控制台是否有网络错误

### Q5: Dashboard 数据为空？

- 运行测试数据生成命令: `python manage.py seed_test_data`
- 确认数据库迁移已执行: `python manage.py migrate`

### Q6: 如何切换小白模式/资深模式？

- 登录页选择模式卡片
- 登录后在控制台小白模式页面点击「切换到资深模式」

### Q7: 平台浏览器拖拽不工作？

- 1688 页面有 X-Frame-Options 限制，iframe 内无法加载
- 使用「跨窗口拖拽」: 从外部浏览器拖拽商品到 ERP 窗口
- 或直接粘贴链接到采集 URL 栏
