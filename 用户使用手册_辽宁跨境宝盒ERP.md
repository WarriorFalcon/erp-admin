## 1. 系统概述

**辽宁跨境宝盒**是一站式跨境电商 ERP 管理系统，覆盖从货源采集到多平台铺货的完整工作流。

### 核心能力

| 模块 | 功能 | 状态 |
|------|------|:--:|
| 商品采集 | 从 1688/淘宝/拼多多 抓取商品信息 | ✅ |
| AI 优化 | 自动生成英文标题、卖点、描述 | ✅ |
| 多平台上架 | 一键铺货至 TikTok/Shopee/Amazon/Temu 等 | ✅ |
| 订单管理 | 多平台订单聚合、发货、取消、导出 | ✅ |
| 库存管理 | 库存列表、预警、同步 | ✅ |
| 物流追踪 | 物流单管理、轨迹查询 | ✅ |
| 利润测算 | 成本×定价倍率 = 利润分析 | ✅ |
| 智能发货 | 自动匹配物流渠+生成运单号+报关 | ✅ |
| 退出登录 | Token 黑名单+本地清除 | ✅ |
| 全球订单分布 | Dashboard 世界地图实时数据 | ✅ |
| AI 助手 | 拓悦AI驱动的智能客服「小辽」 | ✅ |

## 2. 注册与登录

### 2.1 登录步骤

1. 打开系统地址，自动进入登录页
2. **选择模式**: 登录页可选择「🌟 小白模式」或「⚡ 资深模式」
3. 输入手机号 → 点击「获取验证码」→ 填写验证码
   - 开发环境下验证码自动填充
4. 点击「登录」→ 自动跳转控制台

> **测试账号**: `test_user_000` ~ `009`，验证码 `123456`

### 2.2 注册（新用户）

1. 登录页底部点击「立即注册」进入注册页
2. **Step 1**: 填写手机号、验证码、密码、选择用户类型
3. **Step 2**: 实名认证（个人: 身份证 / 企业: 营业执照）
4. **Step 3**: 选择主营类目、目标市场、经营经验
5. **Step 4**: 确认信息 → 勾选用户协议 → 提交申请
6. 等待审核（1-3 工作日），短信通知结果
7. 登录页 → 「审核进度查询」→ 输入手机号查看状态

### 2.3 退出登录

1. 点击右上角头像 → 下拉菜单 →「退出登录」
2. 系统自动清除 Token 并加入黑名单
3. 下次需重新登录

> **安全**: 退出后的 Token 无法再次刷新使用

## 3. 控制台（Dashboard）

### 3.1 小白模式

- **核心指标卡片**: 今日订单、销售额、待发货
- **两大入口**: 「我要上货」「我要发货」
- **待办提醒**: 待处理订单数量

### 3.2 资深模式

完整数据驾驶舱：

- **顶部卡片**: 今日订单量、销售额、客单价、待发货（全部从数据库实时查询）
- **销售额趋势图**: ECharts 折线图
- **世界地图**: 全球订单分布（数据从 `Order.shipping_address.country` 聚合）
  - 点击地区可查看该地区详情
- **平台分布饼图**
- **实时订单流**: 最近 8 条订单
- **交付时效**: 平均交付天数、准时率、在途订单

### 3.3 新手引导

小白模式首次登录自动弹出 3 步向导：
1. 绑定店铺（OAuth 授权）
2. 经营设置（类目/市场/定价模板）
3. 功能解锁确认

## 4. 一站式采集上货（核心功能）

导航: **商品 → 一站式采集上货**

### 4.1 小白模式（粘贴链接 → 一键上架）

1. 复制 1688/淘宝/拼多多 商品链接
2. 粘贴到输入框 → 按 `Ctrl+Enter` 或点击「🚀 开始采集并上架」
3. 系统自动完成：
   - 采集商品信息（标题/图片/价格/规格）
   - AI 合规预检
   - AI 生成英文标题/卖点/描述
   - 利润测算
4. 在决策弹窗中确认 → 点击「确认上架」

### 4.2 资深模式

- **左侧**: 链接输入 + 目标平台多选 + 一键采集按钮
- **右侧**: 商品预览 + AI 优化结果
- **爆品灵感区**: 可展开的热卖推荐

### 4.3 批量模式（资深专属）

1. 点击「批量模式」展开面板
2. 多行链接导入 或 Excel 文件上传
3. 配置铺货规则（目标平台/加价方式/店铺）
4. 点击「开始批量采集并铺货」
5. 查看实时进度和结果

### 4.4 集采模式

采集完成后自动进入集采工作流：
- 步骤3: 选品评估（AI 自动评级）
- 步骤4: 筛选确认（按利润过滤）
- 步骤5: 一键上架（批量提交）

## 8. 订单管理

导航: **订单管理**

### 8.1 订单列表

- 分页展示，支持多维度筛选
- 状态分类: 待处理/已付款/已发货/已签收/已完成/已取消
- 状态统计卡片（点击筛选）

### 8.2 订单操作

| 操作 | 说明 |
|------|------|
| 🚚 发货 | 确认弹窗 → `POST /api/orders/{id}/ship/` |
| ❌ 取消 | 输入取消原因 → `POST /api/orders/{id}/cancel/` |
| 📥 导出 | 下载当前筛选结果 CSV |
| ✏️ 改地址 | 弹窗输入新地址 → `PUT /api/orders/{id}/address` |
| 📝 备注 | 弹窗输入备注 → 保存 |

### 8.3 订单详情

点击「详情」→ 右侧抽屉展示：
- 基本信息（订单号/平台/时间/金额）
- 买家信息
- 商品清单
- 物流信息
- 操作日志时间轴

### 8.4 智能发货

在订单列表页，可调用 `POST /api/orders/smart-ship/`：
- 自动匹配最优物流渠道（cheapest/fastest/safest）
- 生成运单号（LB 开头）
- 自动生成报关信息
- 创建物流记录

## 19. 常见问题（FAQ）

### Q: 登录收不到验证码？
开发模式自动填充验证码 `123456`，无需等待短信。

### Q: AI 生成标题/描述失败？
检查后端健康: `GET /api/health/`。AI 超时 90 秒，请耐心等待。

### Q: 页面空白/按钮灰色？
- 确认后端已启动: `http://127.0.0.1:8000/api/health/`
- 确认前端已启动: `http://localhost:5173`
- 检查浏览器控制台

### Q: Dashboard 数据为空？
运行: `python manage.py seed_test_data --products 25 --orders 80 --shipments 30`

### Q: 如何切换模式？
登录页选择「小白模式」或「资深模式」，控制台顶部可切换。

### Q: 如何调用智能发货？
```js
fetch('/api/orders/smart-ship/', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ order_ids: [1,2], logistics_pref: 'cheapest' })
})
```
