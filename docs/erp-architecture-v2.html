技术选型理由：
• Vue 3 + Composition API：现代响应式，更好的 TypeScript 支持（后续可迁移）
• Element Plus：企业级组件库，与妙手 ERP 风格接近
• ECharts：豪哥要求的销量/库存/采集成功率可视化，功能完善
• Pinia：比 Vuex 更简洁，适合中小型项目
