# 平台授权流程

## 通用流程
1. 调用 `GET /api/auth/<platform>/login/` 获取授权 URL。
2. 用户在平台授权后，平台回调 `GET /api/auth/<platform>/callback/?code=xxx`。
3. 后端使用 `code` 换取 Token，使用 `cryptography.fernet` 加密后保存至 `PlatformToken`。
4. Token 同步写入 Redis 缓存（密文）用于快速读取。
5. Celery 定时任务自动刷新临近过期 Token。

## TikTok Shop 配置（示例）
- 回调地址：`https://your-domain/api/auth/tiktok/callback/`
- 授权入口：`/api/auth/tiktok/login/`
- 刷新入口：`POST /api/auth/tiktok/refresh/`

## Amazon 配置（示例）

- 刷新入口：`POST /api/auth/1688/refresh/`

## 密钥配置
- 环境变量必须设置：
  - `FERNET_KEY`：Fernet 对称加密密钥。
  - `DJANGO_SECRET_KEY`：Django 密钥。

生成 `FERNET_KEY` 示例：
```python
from cryptography.fernet import Fernet
print(Fernet.generate_key().decode())
```
