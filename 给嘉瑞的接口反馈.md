## 3. 问题排查清单

如果接口不通，请嘉瑞检查：

- [ ] **后端服务是否启动？**
  ```bash
  ps aux | grep python
  # 或
  systemctl status your-backend-service
  ```

- [ ] **端口是否正确开放？**
  ```bash
  netstat -tlnp | grep 8000
  ```

- [ ] **防火墙是否放行？**
  - 服务器安全组是否开放 8000 或 443 端口

- [ ] **Nginx 反向代理是否配置？**
  - 如果后端跑在 `:8000`，Nginx 是否做了转发

- [ ] **CORS 跨域是否配置？**
  - 需要允许 `https://erp-admin-tau.vercel.app` 访问
