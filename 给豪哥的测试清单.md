## 一、启动前检查

### 1. 检查后端是否启动
打开浏览器访问：
```
http://localhost:8000/api/health/
```
- ✅ 正常：返回 `{"status": "ok"}`
- ❌ 异常：页面打不开或报错

### 2. 如果后端没启动
**方法A：用Docker镜像（推荐）**
```bash
cd D:\试验场\ERP
docker load < 聊天记录\群聊\backend-latest.tar
docker run -d -p 8000:8000 --name backend backend:latest
```

**方法B：找嘉瑞要最新代码**
让嘉瑞把后端代码同步到GitHub，然后：
```bash
cd backend
python manage.py runserver 0.0.0.0:8000
```
