# 辽宁跨境宝盒 ERP 后端补全任务书

> **致黄嘉瑞**：本文档基于前端现有 API 调用梳理，列出了**后端已有但仅为 Demo(演示) 的接口**和**前端调用但后端完全缺失的接口**，按优先级排列，方便逐一实现。

---

## 一、项目结构速览

```
backend-new/
├── apps/
│   ├── core/              ← 核心业务(认证/商品/订单/库存/物流/AI/报表)
│   ├── creator_mgt/        ← 达人管理
│   ├── sku_mgt/            ← SKU 管理(带 Redis 缓存)
│   ├── task_mgt/           ← 通用任务管理
│   └── selection_engine/   ← 选品引擎(ROAS 测算)
├── myproject/              ← Django 配置
├── .env                    ← 环境变量(已配好)
└── requirements.txt        ← 依赖
```

前端基准路径：`erp-admin/src/api/` 下 18 个封装文件 + 4 个 views 中的直接调用

## 七、环境配置

`.env` 中关键配置（已配好，无需修改）：

| 配置项 | 当前值 | 说明 |
|--------|--------|------|
| `DEMO_MODE` | `false` | 使用真实业务路由 |
| `RBAC_ENFORCE` | `False` | 开发关闭 RBAC |
| `TUOYUE_NEW_API_AUTHORIZATION` | `sk-TP8M...` | 拓跃 AI 密钥 |
| `SMS_PROVIDER` | `aliyun` | 阿里云短信 |
| `CELERY_TASK_ALWAYS_EAGER` | `True` | 同步执行 Celery 任务 |
| `CACHE_FAILOPEN_LOCMEM` | `true` | Redis 不可达时回退本地缓存 |

## 八、实现优先级汇总表

| 优先级 | 接口数 | 说明 |
|--------|--------|------|
| **P0** | ~25 | 商品上货/采集/授权 → 前端核心功能全部依赖 |
| **P1** | ~10 | 库存/物流/上传 → 业务完整链路 |
| **P2** | ~5 | Dashboard/报表/达人 → 体验提升 |
| **P3** | ~2 | 订单备注/选品兼容 → 锦上添花 |
