## 二、开发优先级清单

### 🔴 P0 - 杀手级功能（MVP核心，决定产品能否打动人）

#### 1. 1688一键抓取（最优先级）
**战略价值**：输入链接/图片/条码 → AI全自动 → 2步上货

**前端任务**：
- [ ] 输入框支持3种方式：粘贴1688链接 / 上传商品图片 / 扫码枪扫码
- [ ] 抓取加载动画（骨架屏 + 进度提示）
- [ ] 抓取结果预览（标题/图片/价格/规格）
- [ ] 对接后端1688抓取API（或mock数据）

**涉及文件**：
- 新建 `src/views/goods/QuickCaptureView.vue`

**阻塞依赖**：嘉瑞后端1688抓取接口

---

#### 2. 集采模式开关
**战略价值**：一键切换「采集→选品→上货」闭环 vs 「采集→草稿箱」

**前端任务**：
- [ ] 商品上货页面顶部增加集采模式开关
- [ ] 开关开启：采集商品自动进入选品决策流程
- [ ] 开关关闭：采集商品存入草稿箱
- [ ] 集采模式下增加利润阈值设置（利润额/利润率）

**涉及文件**：
- 修改 `ListingView.vue`
- 新增 `src/composables/useCollectionMode.js`

---

#### 3. 小白模式首页（2个大按钮）
**战略价值**：零学习成本，新手第一步看到的界面

**前端任务**：
- [ ] 新建 `src/views/HomeView_Beginner.vue`
- [ ] 仅展示2个大按钮：「我要上货」「我要发货」
- [ ] 向导式引导（Step 1→2→3→完成）
- [ ] 页面顶部模式切换开关
- [ ] 仅展示3个核心数据：今日订单数、今日销售额、待发货订单

**涉及文件**：
- 新建 `src/views/HomeView_Beginner.vue`
- 修改 `router/index.js`（根据模式加载不同首页）

---

### 🟡 P1 - 核心闭环功能（决定产品能否完整跑通）

#### 4. 订单列表 + 物流智能匹配
**战略价值**：订单归集 + 物流闭环，最高频刚需

**前端任务**：
- [ ] 优化现有订单列表UI
- [ ] 批量选择订单功能
- [ ] 发货偏好选择（最便宜/最快/最稳妥）
- [ ] 物流渠道比价展示（云途/燕文/4PX）
- [ ] 批量打单按钮 + 打印弹窗
- [ ] 物流轨迹追踪视图

**涉及文件**：
- 修改 `src/views/orders/OrderListView.vue`
- 新建 `src/views/logistics/SmartShippingDialog.vue`

**阻塞依赖**：嘉瑞后端订单接口 + 物流API接口

---

#### 5. Dashboard小白模式极简化
**战略价值**：只展示对新手有用的数据

**前端任务**：

#### 6. 爆品推荐5大榜单
**战略价值**：选品决策核心数据源

**前端任务**：

- [ ] 5大榜单Tab切换：热销榜 / 飙升榜 / 高利润榜 / 平台扶持榜 / 辽宁产业带榜
