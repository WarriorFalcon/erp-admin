# 辽宁跨境宝盒 全模块技术性实现详解

## 一、全局技术架构总览

### 基础技术栈锚定
- **前端表现层**：Vue 3.x (Composition API) + Vite 5.x + Element Plus 2.x
- **前端状态层**：Pinia + Vue Router 4 + Axios
- **前端可视化层**：ECharts 5.x + vue-echarts
- **后端接口层**：Django 4.x + DRF
- **后端核心层**：Python 3.10+ + Celery + MySQL 8.0 + Redis
- **AI能力层**：拓岳AI API（兼容OpenAI格式）
- **硬件适配层**：WebUSB / WebBLE / Node.js 原生插件

---

## 二、八大模块技术要点

### 模块1：双模式工作台
**核心**：v-mode指令 + Pinia全局状态，模式切换无刷新无状态丢失
- 复用现有：`src/stores/useAppStore.js`、`src/directives/mode.js`
- 新增：`modeConfig`对象定义两套渲染规则

### 模块2：智能商品全链路上货
**核心**：「输入链接/图片/条码 → AI全自动 → 2步上货」
- 新建：`src/views/goods/OneStopView.vue`
- 复用：`CollectView.vue`采集逻辑、`ListingView.vue` AI生成
- 阻塞：嘉瑞1688抓取接口

### 模块3：专属内置浏览器
**核心**：Chromium内核 + Session隔离 + 拖拽跨iframe通信
- 复用：`src/components/PlatformBrowser.vue`
- 新增：反向代理Session隔离、postMessage通信

### 模块4：订单与物流全闭环
**核心**：多平台订单归集 + Agent审单 + 物流智能匹配
- 复用：`src/views/orders/`
- 新增：物流比价UI、智能打单
- 阻塞：嘉瑞订单API + 物流商API

### 模块5：卖家之家原生集成
**核心**：API原生嵌入，非iframe嵌套
- 复用：现有商品/合规/物流组件
- 阻塞：泽华卖家之家API

### 模块6：极简实效数据驾驶舱
**核心**：砍花哨图表，只留决策必需数据
- 小白：3卡片 + 1趋势图 + AI解读
- 资深：全维度 + 利润实时核算
- 复用：`src/views/DashboardView.vue`

### 模块7：Agent工作流引擎
**核心**：状态机 + Celery + 拓岳AI调度
- 新建：`src/views/workflow/`（资深模式）
- 小白：仅5个标准化模板一键启用

### 模块8：龙虾硬件深度集成
**核心**：WebUSB/WebBLE + 硬件级加密
- 新建：`src/utils/hardware.js`
- 阻塞：E:\u-claw 硬件SDK

---

## 三、关键技术规范

1. **双模式渲染**：`v-mode`指令 + Pinia全局状态
2. **全链路数据闭环**：采集→校验→处理→存储→同步→展示
3. **AI能力复用**：`src/api/ai.js`统一封装
4. **第三方API适配**：后端适配层，前端仅调聚合接口
5. **硬件通信规范**：`src/utils/hardware.js`统一封装
